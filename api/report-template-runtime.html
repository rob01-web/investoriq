Under the Downside scenario, the investment remains cash-flow positive with returns modestly below institutional targets, demonstrating meaningful downside protection. The Base Case produces returns squarely within the target range for value-add multifamily strategies, while the Upside scenario illustrates the sensitivity of equity returns to rent growth and exit pricing once stabilization is achieved.

Section 17 Discounted Cash Flow (DCF) Intrinsic valuation via discounted cash flow (5-year hold)

The DCF analysis estimates the intrinsic value of the property by projecting annual net cash flows and discounting them at a risk-adjusted rate reflecting investor return requirements. This method is preferred by institutional underwriters because it reveals the true economic value of an investment independent of market noise or temporary performance fluctuations.

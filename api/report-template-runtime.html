The DCF analysis estimates the intrinsic value of the property by projecting annual net cash flows and discounting them at a risk-adjusted rate reflecting investor return requirements. This method is preferred by institutional underwriters because it reveals the true economic value of an investment independent of market noise or temporary performance fluctuations.
